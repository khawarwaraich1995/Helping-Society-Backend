Client Authentication

API endpoints for managing clients authentication

Register Endpoint

This endpoint allows you to register client and generate token for authentication. Along token client details will be displayed. To call authorized endpoints you need access_token generated by this endpoint.

Example request:
curl --request POST \
    "http://helping-society.test/api/v1/client/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"John Panda\",
    \"email\": \"john@example.com\",
    \"password\": \"secret\"
}"

const url = new URL(
    "http://helping-society.test/api/v1/client/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "John Panda",
    "email": "john@example.com",
    "password": "secret"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://helping-society.test/api/v1/client/register',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'name' => 'John Panda',
            'email' => 'john@example.com',
            'password' => 'secret',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Example response (200):

    "http://helping-society.test/api/v1/client/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \

    \"email\": \"testuser@example.com\",
    \"password\": \"secret\"

const headers = {

fetch(url, {

Logout Endpoint

This endpoint allows you to remove client token generated for authentication to logout user. To logout token must be required in header otherwise client will be Unauthenticated.

Example request:
curl --request POST \
    "http://helping-society.test/api/v1/client/logout" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Authorization: Bearer $token"
const url = new URL(
    "http://helping-society.test/api/v1/client/logout"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Authorization": "Bearer $token",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://helping-society.test/api/v1/client/logout',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
            'Authorization' => 'Bearer $token',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Example response (200):


{
    "status": true,
    "message": "You have successfully logged out!"
}

    "message": "Unauthenticated."
